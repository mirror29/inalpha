# 分市场信息源手册

按用户提及的市场选信息源体系。这里的"查"动作都经 web.search / web.search_news 检索公开披露，财务数字经 data.get_fundamentals 验证。市场清单非穷举——用户提到未列出的市场时，按"当地交易所披露 + 监管文件 + 公司 IR"同构推断。

## 美股
- 一手：SEC 10-K / 10-Q / 8-K / S-1 / S-3 / Form 4、业绩会记录、投资者材料、客户与供应商的对方披露、专利与标准组织
- 重点核：shelf / ATM / 可转债 / SBC / 内部人卖出、客户集中度、backlog 与收入结构、毛利与稼动率、卖方预期差

## A股
- 一手：年报 / 半年报 / 季报 / 临时公告、交易所问询函与监管函、互动易 / 上证 e 互动、招投标与中标公告、客户验厂认证、环评 / 能评 / 地方项目备案、专利标准、海关数据、上下游上市公司交叉验证
- 重点核：应收 / 存货 / 合同负债 / 经营现金流、毛利率与产能利用率与在建转固、关联交易与客户真实性、定增 / 可转债 / 股权质押 / 商誉、补贴与订单商业性、主题炒作后的估值压力

## 港股
- 一手：HKEX 披露易、年报中报、配售 / 可换股 / 关连交易公告、公司演示材料、业务重心在内地时的内地监管文件
- 重点核：流动性与价差、再融资压力、关连方治理、港股通资格、内地政策敞口

## 台股
- 一手：MOPS 公开资讯观测站、**月营收**（独有高频信号）、IR 简报、客户供应商交叉验证、行业刊物
- 重点核：月营收拐点、客户集中度、汇率敏感度、两岸地缘风险、产能与认证排程

## 日股
- 一手：TDnet、决算材料、统合报告书、分部数据、行业期刊
- 重点核：保守指引惯性、汇率敏感、交叉持股与治理改革、覆盖度低带来的认知差

## 韩股
- 一手：DART、出口统计（高频信号）、IR 材料、客户生态披露
- 重点核：大客户依赖、存储周期敞口、汇率与地缘、散户主题波动、资本开支周期时点

## 欧股
- 一手：当地交易所披露、年报与 ad hoc 公告、EU 补助 / 项目文件、客户合作公告、行业期刊与标准组织
- 重点核：流动性、折算风险、政府补助依赖、被并购可能性、专业覆盖缺口
