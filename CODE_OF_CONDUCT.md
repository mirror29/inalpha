# 行为准则 / Code of Conduct

> 本项目采用 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 行为准则。
> This project adopts the Contributor Covenant v2.1 Code of Conduct.

## 我们的承诺

为了营造开放、友好的环境，作为贡献者和维护者，我们承诺让参与本项目和社区的每个人都不会因为年龄、体型、可见或不可见的残疾、族裔、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人形象、种族、宗教信仰或性认同与取向而受到骚扰。

我们承诺以促进开放、欢迎、多元、包容和健康社区的方式行事和互动。

## 我们的标准

**有助于营造积极环境的行为示例**：

- 对他人表达同理与善意
- 尊重不同的意见、观点与经验
- 给予并优雅地接受建设性反馈
- 为我们的错误承担责任并向受影响者道歉，并从中学习
- 关注的不只是个人最佳利益，而是整个社区的最佳利益

**不可接受的行为示例**：

- 使用性化语言或图像，以及任何形式的性关注或挑逗
- 嘲讽、侮辱或贬低性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人私人信息，如住址或电子邮件
- 在专业场合中合理判断为不当的其他行为

## 执法责任

社区维护者负责澄清并执行我们的可接受行为标准。对于他们认为不当、具有威胁性、攻击性或有害的任何行为，社区维护者有权利和责任移除、编辑或拒绝评论、提交、代码、wiki 编辑、issue 和其他贡献。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共空间代表项目或其社区时。

## 执法

辱骂、骚扰或其他不可接受行为可通过 [`SECURITY.md`](SECURITY.md) 中列出的渠道向维护者报告。所有投诉将被快速、公平地审查与调查。

所有社区领导者都有义务尊重任何事件报告者的隐私与安全。
